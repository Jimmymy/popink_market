About

Japan-focused B2B tattoo supply commerce demo.

Popink Market is built as a Medusa + Next.js ecommerce prototype for tattoo studios, artists, and professional buyers. This GitHub Pages version mirrors the storefront layout as a static preview.

  • Japan region and JPY pricing direction.
  • Professional tattoo supply category structure.
  • Checkout and Admin flows remain in local or staging environments.
  • Future B2B account review and wholesale pricing path.

Wholesale

Wholesale workflows are planned as a later B2B layer.

The current public preview keeps catalog browsing simple. Customer approval, account-specific price visibility, and quotation workflows are intentionally not active on GitHub Pages.

Shipping

Japan shipping options are validated in the full demo.

The local Medusa environment currently validates a Standard Japan Shipping path. Real carrier integration belongs to staging and production planning.

Returns

Return policy content is staged for production review.

This static page preserves the information architecture while legal, operations, and product-specific return rules are finalized.

Contact

Contact forms are not connected on GitHub Pages.

Live form submission, account support, and order communication require the backend services that are not executed in this static preview.

Account

Account login is disabled in this static preview.

Use the local or staging storefront for customer registration and login verification.

Cart

Your cart is empty in the static preview.

Cart, checkout, payment fallback, and order submission require Medusa Store API.
